# Study English — AI英会話練習アプリ

マイクに向かって英語を話すと、AIが英語で返答しながら文法・表現の改善点を日本語でフィードバックしてくれる英会話練習アプリです。

## 何ができるか

- **トピック選択**: AIが提案するシーンから練習したい状況を選ぶ
- **音声会話**: マイクボタンを押しながら英語を話す
- **AIの英語返答**: GPT-4oが自然な英語で会話を続ける（音声読み上げあり）
- **日本語フィードバック**: 毎回の返答と一緒に以下を表示
  - 文法ミスの指摘と正しい表現
  - より自然なネイティブ表現の提案
  - 高校生レベルでシンプルに言える代替表現

## 技術スタック

| 用途 | 技術 |
|------|------|
| フレームワーク | Next.js 15 (App Router) |
| UI | shadcn/ui + Tailwind CSS |
| 音声認識 (STT) | OpenAI Whisper |
| 会話AI | OpenAI GPT-4o |
| 音声合成 (TTS) | OpenAI TTS |
| デプロイ | Vercel |

## セットアップ

### 1. 依存パッケージをインストール

```bash
npm install
```

### 2. 環境変数を設定

`.env.example` を `.env.local` にコピーして、OpenAI APIキーを入力します。

```bash
cp .env.example .env.local
```

`.env.local` を開いて編集：

```env
OPENAI_API_KEY=sk-xxxxxxxxxxxxxxxx
```

> APIキーは [OpenAI Platform](https://platform.openai.com/api-keys) で取得できます。

### 3. 開発サーバーを起動

```bash
npm run dev
```

ブラウザで [http://localhost:3000](http://localhost:3000) を開く。

## 使い方

1. トップ画面でトピックを選ぶ（例：「カフェで注文する」「旅行の計画を話す」）
2. 会話画面が開いたら、AIが最初の一言を英語で話しかけてくる
3. マイクボタンを**押しっぱなし**にして英語で話す
4. ボタンを離すと音声がAIに送られ、英語の返答 + 日本語フィードバックが表示される
5. AIの音声読み上げを聞きながら会話を続ける

## Vercelへのデプロイ

```bash
vercel
```

Vercelのダッシュボードで環境変数 `OPENAI_API_KEY` を設定することを忘れずに。

| Whisper | 話した時間に応じて（$0.006/分） |
| GPT-4o | 送受信したテキスト量に応じて |
| TTS | 読み上げた文字数に応じて |
